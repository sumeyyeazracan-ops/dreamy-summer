# ✨ Dreamy Summer - Soft Girl Website

🌙 Rüya gibi bir yaz yaşa! Kız kulübü teması ile diziye benzer bir website.

## 🎨 Özellikler

- **Naif Soft Girl Estetik**: Toz pembe, lavanta, baby blue renkler
- **Yaz Dizisi Atmosferi**: Hafif, tatlı, romantik tema
- **Duyarlı Tasarım**: Mobil ve masaüstü uyumlu
- **Etkileşimli Bileşenler**: Smooth scroll, animasyonlar, hover efektleri
- **Weather Dashboard**: Gerçek zamanlı hava durumu bilgileri
- **Kolay Özelleştirme**: HTML, CSS, JavaScript ile kolayca düzenlenebilir

## 🚀 Dosyalar

- **index.html** - Ana sayfa
- **weather.html** - Hava durumu dashboard'u
- **style.css** - Ana sayfa tasarımı
- **weather-style.css** - Weather dashboard tasarımı
- **script.js** - Ana sayfa interaktivitesi
- **weather-script.js** - Weather dashboard işlevleri
- **README.md** - Kullanım kılavuzu

## 🌤️ Weather Dashboard

- 🌍 Konum tabanlı hava durumu
- 🔍 Şehir arama
- 📊 Detaylı hava bilgileri (nem, rüzgar, basınç, görüş, UV indeksi)
- 📅 5 günlük tahmin
- 💡 Hava durumuna göre ipuçları
- 🎨 Soft girl teması ile tasarlanmış

### API Kullanılan:
- **Geocoding API**: Şehir koordinatlarını bulma
- **Open-Meteo Weather API**: Gerçek zamanlı hava durumu verileri

## 🎯 Nasıl Kullanılır?

1. Dosyaları indir
2. `index.html` dosyasını tarayıcıda aç
3. Weather dashboard'a gitmek için "Weather Dashboard'a Git" butonuna tıkla
4. Şehir adı yazarak hava durumunu öğren

## 💻 Teknolojiler

- HTML5
- CSS3 (Gradients, Backdrop Filters, Grid, Flexbox)
- Vanilla JavaScript
- Open-Meteo API
- Geocoding API

## 🌙 Özelleştirme

### Renkleri Değiştir
`style.css` ve `weather-style.css` içindeki `:root` bölümünde renkleri düzenle:
```css
:root {
    --primary-pink: #ffc0d9;
    --accent: #ff69b4;
    /* vs. */
}
```

### İçerik Ekle
HTML dosyalarında metin ve görselleri değiştir.

### Font Değiştir
CSS dosyalarında `font-family` özelliğini güncelle.
