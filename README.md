# ✨ Dreamy Summer - Soft Girl Website

🌙 Rüya gibi bir yaz yaşa! Kız kulübü teması ile diziye benzer bir website.

## 🌸 Özellikler

- **Naif Soft Girl Estetik**: Toz pembe, lavanta, baby blue renkler
- **Yaz Dizisi Atmosferi**: Hafif, tatlı, romantik tema

- **Kolay Özelleştirme**: HTML, CSS, JavaScript ile kolayca düzenlenebilir

## 🚀 Başlangıç

1. Repository'yi clone et
2. `index.html` dosyasını tarayıcıda aç
3. Özelleştir ve kendi içeriğini ekle!

## 📝 Dosyalar

- `index.html` - HTML yapısı
- `style.css` - Tasarım ve stiller
- `script.js` - Etkileşimli özellikler

## 💕 Nasıl Özelleştirilir?

### Renkleri Değiştir
`style.css` içindeki `:root` bölümünde renkleri düzenle:
```css
:root {
    --primary-pink: #ffc0d9;
    --accent: #ff69b4;
    /* vs. */
}
```

### İçerik Ekle
`index.html` içinde metin ve görselleri değiştir.

### Font Değiştir
`style.css` içinde `font-family` özelliğini güncelle.
